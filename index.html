McLaren Applied ATLAS Display API.

The ATLAS Display API facilitates the creation of bespoke ATLAS plugins.

Sample code and best practices are available on our GitHub repository. This API will provide common classes and interfaces for interacting and extending ATLAS 10.

Scope This API provides a common vocabulary for extending and interacting with ATLAS 10. Extension points will include:

  • Automation
  • Displays
  • Tool Windows
  • Session Management
  • Parameter Management
  • Session Navigation

This Automation API provides a set of services for interacting with ATLAS 10.

Services include:

  • Application Service: IApplicationService
  • Diagnostics Service: IDiagnosticsService
  • Display Service: IDisplayService
  • PDA Service: IParameterDataAccessService
  • Page Service: IPageService
  • Recorder Service: IRecorderService
  • Session Service: ISessionService
  • Set Service: ISetService
  • Summary Display Service: ISummaryDisplayService
  • Timebase Service: ITimeBaseService
  • Tracer Service: ITracerService
  • Update Loop Statistics Service: IUpdateLoopStatisticsService
  • Workbook Service: IWorkbookService

This API provides a set of clients for accessing these services.

Clients include:

  • Application Client: ApplicationServiceClient
  • Diagnostics Client: DiagnosticsServiceClient
  • Display Client: DisplayServiceClient
  • Page Client: PageServiceClient
  • PDA Client: ParameterDataAccessServiceClient
  • Recorder Client: RecorderServiceClient
  • Session Client: SessionServiceClient
  • Set Client: SetServiceClient
  • Summary Display Client: SummaryDisplayServiceClient
  • Timebase Client: TimeBaseServiceClient
  • Tracer Client: TracerServiceClient
  • Update Loop Statistics Client: UpdateLoopStatisticsServiceClient
  • Workbook Client: WorkbookServiceClient

These services return data transfer objects (DTOs).

DTOs include:

  • Color: Color
  • Composite Session: CompositeSession
  • Display: Display
  • Display Parameter: DisplayParameter
  • Lap: Lap
  • Page: Page
  • Parameter: Parameter
  • Parameter Values: ParameterValues
  • Recorder: Recorder
  • Session: Session
  • Session Detail: SessionDetail
  • Session Event: SessionEvent
  • Session Id: SessionId
  • Set: Set
  • Statistics Data: StatisticsData
  • Time Base: TimeBase
  • Tracer Category: TracerCategory
  • Tracer Settings: TracerSettings
  • Update Loop Statistics: UpdateLoopStatistics
  • Updateable Statistics: UpdateableStatistics
